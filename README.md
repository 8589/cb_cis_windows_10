# CIS Windows 10 Cookbook

Chef Cookbook for applying CIS Benchmark settings to Windows 10 systems

## Requirements

### Platform:
* Windows 10

### Cookbooks
* windows (>= 4.0)
* windows-security-policy (>= 0.3.7)
* windows_firewall (>= 3.0)
